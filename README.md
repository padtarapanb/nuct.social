# NUCT PR Ops

ระบบจัดการงานประชาสัมพันธ์ชมรม — Dashboard / ปฏิทิน / ทีม / งาน ในไฟล์เดียว
สร้างด้วย React + Vite + Tailwind CSS ข้อมูลทั้งหมดเก็บไว้ใน localStorage ของเบราว์เซอร์ (ยังไม่เชื่อมฐานข้อมูลกลาง)

## รันบนเครื่องตัวเอง

```
npm install
npm run dev
```

เปิด http://localhost:5173

## Deploy ขึ้น Vercel

วิธีที่ 1 — ผ่านเว็บ (ไม่ต้องลงอะไรเพิ่ม)
1. อัปโหลดโฟลเดอร์นี้ขึ้น GitHub (สร้าง repo ใหม่ แล้ว push โค้ดเข้าไป)
2. เข้า https://vercel.com → New Project → เลือก repo นี้
3. Vercel จะตรวจพบว่าเป็น Vite โปรเจกต์เองอัตโนมัติ (Build Command: `npm run build`, Output: `dist`) กด Deploy ได้เลย

วิธีที่ 2 — ผ่านคำสั่ง (เร็วกว่า ไม่ต้องใช้ GitHub ก็ได้)
```
npm install -g vercel
vercel login
vercel --prod
```
ระบบจะถามชื่อโปรเจกต์ แล้วจะได้ลิงก์ประมาณ `https://ชื่อโปรเจกต์.vercel.app`

## โครงสร้างไฟล์
```
src/
  store.js          ข้อมูลเริ่มต้น + logic บันทึก/แก้ไข (localStorage)
  App.jsx            หน้าหลักที่รวม 4 แท็บ
  components/
    Nav.jsx           แถบเมนูด้านบน/ล่าง
    EventPill.jsx      ป้ายงานสีตามประเภทในปฏิทิน
    EventModal.jsx      หน้าต่างแก้ไขงาน + เช็คลิสต์
  pages/
    Dashboard.jsx      หน้าหลัก: งานประจำสัปดาห์ / งานใกล้ถึง / คนไม่ว่าง / คนอยู่พิษณุโลก
    Calendar.jsx       ปฏิทินรายเดือน สีตามประเภทงาน กดวันที่เพื่อเพิ่มงาน
    Team.jsx           รายชื่อทีม เพิ่ม/ลบ/แก้ไขได้
    Tasks.jsx          รวมเช็คลิสต์ทุกงานเป็นลิสต์เดียว กรองตามสถานะ
```

## ขั้นต่อไปที่แนะนำ
- เชื่อมกับ Google Sheets หรือฐานข้อมูลจริง (เช่น Supabase) แทน localStorage เพื่อให้ทุกคนในทีมเห็นข้อมูลเดียวกัน แจ้งได้ถ้าอยากให้ช่วยต่อส่วนนี้
- เพิ่มระบบ login ง่ายๆ กันคนนอกทีมเข้ามาแก้ข้อมูล
